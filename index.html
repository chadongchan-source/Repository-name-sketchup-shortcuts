💡
꿀팁: 단축키는 영문 입력 상태에서만 작동해요! 한글로 바꾸면 안 먹히니까 작업 전 꼭 확인하세요.
⬛ 선택
🖱️
선택 도구
객체를 클릭해서 선택해요. 더블클릭하면 그룹 안으로 들어가요.
Space
🔲
전체 선택
화면의 모든 객체를 한 번에 선택해요.
Ctrl+A
❌
선택 해제
선택된 것을 모두 해제해요. 빈 공간 클릭해도 돼요.
Ctrl+D
✏️ 그리기
📏
선 그리기
직선을 그려요. 숫자 입력 후 Enter로 정확한 길이 지정 가능!
L
⬜
사각형 그리기
사각형을 그려요. "500,300" 입력하면 정확한 크기로 그려져요.
R
⭕
원 그리기
원을 그려요. 반지름 숫자 입력 후 Enter.
C
🌙
호(곡선) 그리기
곡선 형태의 호를 그려요. 창문 상단 아치 등에 활용!
A
⬡
다각형 그리기
다각형을 그려요. 숫자로 면 개수 지정 가능.
Alt+P
🔧 수정
📦
밀기/당기기 (Push/Pull)
면을 클릭하고 밀거나 당겨서 3D 형태로 만들어요. 목공 모델링의 핵심!
P
↔️
이동
선택한 객체를 옮겨요. Ctrl 누르면 복사+이동!
M
🔄
회전
객체를 회전시켜요. 각도 숫자 입력 후 Enter로 정확하게!
Q
📐
크기 조절
객체 크기를 늘리거나 줄여요.
S
🔲
간격 띄우기 (Offset)
선택한 면이나 선에서 일정 간격 떨어진 복사본을 만들어요.
F
🧹
지우기
선이나 면을 지워요. Shift+E는 부드러운 선 토글.
E
🎨
페인트 통 (재질 적용)
면에 재질이나 색상을 칠해요. 목재 텍스처 적용할 때 자주 써요.
B
📏
줄자 / 가이드라인
거리를 측정하거나 가이드 선을 만들어요. 치수 확인에 필수!
T
↩️
실행 취소
방금 한 작업을 되돌려요. 실수했을 때 바로 눌러요!
Ctrl+Z
↪️
다시 실행
취소한 작업을 다시 해요.
Ctrl+Y
👁️ 보기·탐색
🌐
화면 회전 (Orbit)
3D 화면을 돌려가며 볼 수 있어요. 마우스 휠 누른 채 드래그해도 돼요.
O
✋
화면 이동 (Pan)
화면을 좌우상하로 이동해요. Shift+휠 드래그도 가능.
H
🔍
확대/축소 (Zoom)
화면을 확대하거나 줄여요. 마우스 휠 스크롤이 더 편해요.
Z
🖼️
전체 화면에 맞추기
모든 객체가 화면에 딱 맞게 보이도록 자동으로 맞춰줘요.
Ctrl+Shift+E
🎯
선택 항목 확대
선택한 객체만 화면 가득 보여줘요.
F5
📷
투시/평행 투영 전환
원근감 있는 투시도 ↔ 정면도(직교) 전환. 도면 작업 시 평행으로!
Camera메뉴
🩻
X-Ray 모드 (투명하게 보기)
모든 면이 투명하게 보여서 내부 구조 확인에 유용해요.
Alt+X
💾 파일·저장
💾
저장
작업 내용을 저장해요. 자주 누르는 습관 필수!
Ctrl+S
📂
다른 이름으로 저장
새 이름이나 다른 경로에 저장해요.
Ctrl+Shift+S
📄
새 파일
새 스케치업 파일을 열어요.
Ctrl+N
📁
파일 열기
기존 .skp 파일을 불러와요.
Ctrl+O
🖨️
인쇄
현재 화면을 인쇄해요. PDF로 내보내기도 가능!
Ctrl+P
🍯 알면 편한 꿀팁
💡
정확한 치수 입력: 도구 선택 후 드래그하면서 숫자를 바로 타이핑하면 정확한 수치로 작업돼요. (예: 밀기/당기기 중 "2400" 입력 → 2400mm)
🔵
색상 축으로 방향 잡기: 빨강=X축, 초록=Y축, 파랑=Z축! 선이 해당 색으로 바뀌면 그 방향으로 이동 중이에요.
📦
그룹 만들기: 작업 완료 후 선택 → 오른쪽 클릭 → "그룹 만들기" 꼭 하세요! 안 하면 다른 객체랑 붙어버려요.
🖱️
마우스 휠 활용: 스크롤=줌 / 가운데 버튼 누른 채 드래그=Orbit / Shift+가운데 드래그=Pan
